# YQRefresh
下拉刷新上拉加载更多

## Features
1. 支持下拉刷新和上拉加载
2. 可随意自定义动画效果

## Installation

### Carthage
```
github "yuyedaidao/YQRefresh"
```

### CocoaPods
```
pod 'YQRefresh', :git => 'https://github.com/yuyedaidao/YQRefresh.git'
```
